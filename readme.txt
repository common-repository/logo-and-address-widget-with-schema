== Short Description ==
Widget to add your company logo, address, and business hours with schema.

== Description ==
Are you still trying to show your business address in the plain text widget? Are you tired of adding custom CSS to make your business hours look good on your website? No more worries! Here is a widget, which will help you to add your business address and hours (if you want) as a widget with schema in less than 5 minutes.

The widget takes care of all the formatting. You can change labels, show hours for Monday through Friday or each day separately. We've even included schema markup, which is great for SEO.

Features List:
- Upload your logo and even specify alternate text for your logo. Use the WordPress Media Library to upload/select your logo.
- Add your full address, email, phone, and fax number.
- You can either show labels or show icons for the address, email, phone and fax number.
- Enable or disable schema for the address.
- Show business hours by weekdays and for weekends.

== Installation ==

Option 1:

1. Install the plugin in the Dashboard => Plugins menu => Add New => Upload => Select the zip file => Install Now
2. After plugin activation, please go to Dashboard => Appearance => Widgets
3. Choose the "Logo and Address Widget with Schema" widget then drag and drop it to any sidebar or footer
4. Enter your contact information and click the Save button

Option 2:

1. Unzip the zip file you've downloaded.
2. Upload the "logo-and-address-widget-with-schema" folder via FTP to your /wp-content/plugins/ folder/directory
3. Activate the "Logo and Address Widget with Schema" plugin by going to Dashboard => Plugins and clicking Activate next to the plugin


GETTING STARTED

Here is more information about the plugin.

1. After activating the plugin, please go to Appearance => Widgets
2. Please select the "Logo and Address Widget with Schema" widget then drag and drop it your desired sidebar.
3. Enter your contact information and save the widget to see it on your website.


== Frequently Asked Questions ==

= Is schema for the address added by default? =
Yes, it is enabled by default. If you have used any other plugin to add schema for address, please disable it and this plugin will take care of schema or you can keep it enabled and disable it in this plugin.

= How do I use this plugin to show multiple addresses? =
Drag and drop the "Logo and Address Widget with Schema" widget in the same or into another sidebar and then configure it to show the next address.

= How do I display operational hours? =
Enable the Business Hours section, then choose the "Mon-Fri" option by checking the checkbox then enter your business hours. If you are open on weekends, you can specify those hours as well.
